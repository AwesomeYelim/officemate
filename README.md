# OfficeMate — 지란지교 해커톤 2026 제출물

OfficeMate는 오피스챗에 붙는 AI 매칭 어시스턴트로, 동료 간 식사·번개 모임을 자동으로 개설·매칭·정산합니다. 이 리포는 해커톤 제출물(랜딩 데모, 사업계획서, 화면 목업, HR 콘솔 시뮬레이터)을 정적 사이트로 묶은 것입니다.

## 페이지 목록

- `index.html` — 제출물 허브 (시작 페이지)
- `landing.html` — 랜딩페이지 + 라이브 데모 (`#demo` 앵커로 데모 섹션 진입)
- `biz-plan.html` — 사업계획서
- `flow-mockup.html` — 핵심 매칭 플로우 목업 (7화면)
- `journey-mockup.html` — 사용자 여정 목업 (5화면)
- `hr-console.html` — HR 콘솔 시뮬레이터

## 배포

Vercel에 연결된 저장소이며, `main` 브랜치에 push하면 자동으로 배포됩니다.
